This project is a single-page website dedicated to the impact of SVG on the future web design.



  • SVG: What is it?

    This website introduces some of the basic features and concepts of SVG, along with examples and links to show
    how it can revolutionize web design.
    Scalable Vector Graphics (SVG) is an open-sourced XML-based vector image format that can also provide a platform for 2-D graphics. SVG graphics are versatile, user-friendly, and have excellent presentation capabilities.

    There is no loss of clarity when SVG images are scaled up or down and no need to create separate images for high-density screens, which for a developer means: write once, deploy anywhere!! It is supported by ALL modern browsers, check at
    Can I Use?, a compatability chart for browsers and OS. Mobile SVG is also increasing in popularity becoming the
    preferred graphics platform for the mobile phone industry.

  • Why use SVG?

    Smooth and Dynamic Animations


    SVG animations can be created using a simple text-editor and can be reused and gzipped (compressed), this improves page load times and performance. The file sizes are smaller because SVG code is a "description" of the graphics, describing the lines, curves and shapes. etc.

    SVG builds upon many other standards including the DOM for scripting and interactivity, as well as building upon
    or with SMIL, CSS3 & Javascript.

    SVG truly offers something for everyone! It is recommended by the WAI-ARIA for making websites more accessible to people with disablities.
    SVG graphics and animations are accessible to assistive technologies(AT), such as screen readers and speech recognition tools.
    It also can be resized proportionately with the graphical content intact for the benefit of anyone with less than 20/20 vision.
    ****** SVG looks crisp on any resolution and offers responsive images for responsive web design ******
